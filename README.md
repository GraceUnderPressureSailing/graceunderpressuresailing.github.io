# Grace Under Pressure Sailing

Source for [graceunderpressuresailing.github.io](https://graceunderpressuresailing.github.io/).

The site is a Markdown/Jekyll record of practical boat-system projects, field
observations and passage work aboard *Grace Under Pressure*.

## Publishing

GitHub Pages builds the site from the `main` branch. Most content can be edited
as ordinary Markdown; `_config.yml` controls the site-wide navigation and
metadata.

Project source code remains in its own repository. The website contains the
narrative, results and links back to the relevant technical repository.
